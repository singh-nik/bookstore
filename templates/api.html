Bookshop Api

User will be able to get data from this book review website to use another application. Data will provided in JSON(Java Script Object Notation) formate. To get data user make a GET request to this website’s (/api/isbn) route, where [isbn] is an ISBN number of the book, it return a JSON response containing the book’s title, author, publication date, ISBN number, review count, and average score. The resulting JSON should follow the format:
{
"title": "Memory",,
"author": "Doug Lloyd",
"year": 2015,
"isbn": "1632168146",
"review_count": 28,
"average_score": 5.0
}
